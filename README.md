# MOSTRAMUN'26 Website Draft (Mostra style)

A static single-page website draft for MOSTRAMUN'26.

## Files
- `index.html` – page structure
- `styles.css` – styling (black + green `#45B17A` palette)
- `script.js` – countdown, committees grid, modal, mobile nav

## Run locally
Just open `index.html` in a browser.

## Quick edits
- **Countdown date (15 May):** `script.js` → `getTargetDate()`
- **Hero photo:** replace the `.hero-media` placeholder in `index.html` or set a background image in `styles.css`
- **Committee details/abouts:** update `committeeData` in `script.js`
- **Apply buttons/forms:** replace placeholder buttons with form links
